# DPO2U Midnight Relayer

> This project is built on the Midnight Network.

Cross-chain compliance relay for Midnight Network with ZK proofs. Enables privacy-preserving compliance attestation bridging between Midnight and other chains.

## Features

- Cross-chain compliance verification relay
- Zero-knowledge proof validation
- Privacy-preserving attestation bridging
- Compact smart contract integration

## Tech Stack

- **Blockchain:** Midnight Network
- **Smart Contracts:** Compact
- **Privacy:** Zero-Knowledge Proofs
